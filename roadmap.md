Passo 2 - Criar um sistema gerenciador de loja de varejo com gestão de estoque de produtos

✓ SGLV 🛒 

##### Requisitos

###### 1\. Cadastro de Produtos
1. ✓ Nome do Produto: Campo para armazenar o nome do produto.
2. ✓ Código do Produto: Um código único para identificar o produto.
3. ✓ Categoria: Categorias como "eletrônicos", "vestuário", etc.
4. ✓ Quantidade em Estoque: Campo para armazenar a quantidade disponível no estoque.
5. ✓ Preço: Preço de venda do produto.
6. ✓ Descrição: Detalhes adicionais sobre o produto.
7. ✓ Fornecedor: Informações sobre o fornecedor do produto.

###### 2\. Gestão de Estoque
1. ✓ Adicionar ao Estoque: Funcionalidade para aumentar a quantidade de um produto no estoque.
2. ✓ Remover do Estoque: Funcionalidade para reduzir a quantidade de um produto no estoque.
3. ✓ Atualização de Estoque: Possibilidade de ajustar manualmente a quantidade de produtos.
4. ✓ Alerta de Estoque Baixo: Sistema de notificação quando a quantidade de um produto atinge um nível mínimo predefinido.

###### 3\. Vendas
1. ✓ Registro de Vendas: Função para registrar a venda de produtos.
2. ✓ Atualização Automática do Estoque: Ao registrar uma venda, a quantidade em estoque deve ser automaticamente reduzida.
3. ✓ Emissão de Recibo: Geração de um recibo ou nota fiscal após a venda.
4. ✓ Descontos e Promoções: Função para aplicar descontos específicos ou promoções a produtos.

###### 4\. Relatórios
1. ✓ Relatório de Vendas: Relatório detalhado das vendas realizadas, incluindo data, produtos vendidos, quantidade e valor total.
2. ✓ Relatório de Estoque: Visualização da quantidade atual de todos os produtos no estoque.
3. ✓ Histórico de Movimentações: Registro de todas as adições e remoções de estoque.

###### 7\. Integração e Backup
1. ✗ Backup Automático: Funcionalidade para criar backups automáticos do banco de dados.
2. ✗ Importação/Exportação de Dados: Ferramentas para importar e exportar dados, como listas de produtos ou relatórios.

###### 8\. Documentação e Suporte
1. ✗ Manual do Usuário: Documentação clara e acessível explicando como utilizar cada funcionalidade do sistema.
2. ✗ Suporte Técnico: Canal de suporte para resolução de problemas técnicos.
